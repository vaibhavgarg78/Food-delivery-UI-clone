# Swiggy Clone

A responsive Swiggy-inspired frontend clone built using HTML and Tailwind CSS.

## Features
- Responsive Design
- Modern UI
- Tailwind CSS Styling
- Food Categories Section
- Restaurant Cards
- Footer Section

## Tech Stack
- HTML
- Tailwind CSS
